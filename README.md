## Overview

## Getting Started

### Prerequisites

- Node.js (14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd <repository-name>
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

### Available Scripts

In the project directory, you can run:

- **Start the app:**

  ```bash
  npm start
  ```

- **Build for production:**

  ```bash
  npm run build
  ```

- **Run tests:**

  ```bash
  npm test
  ```

- **Eject (if needed):**

  ```bash
  npm run eject
  ```
